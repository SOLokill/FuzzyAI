# FuzzyAI
SKHU - 사귈 확률 퍼지 AI

### **To Run**  
```Shell
# Clone
git clone https://github.com/minjune8506/FuzzyAI/ FuzzyAI
cd FuzzyAI

# Main Program
python3 main.py

# Draw Graph
python3 graph.py
```
